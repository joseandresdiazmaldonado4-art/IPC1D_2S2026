# Instrucciones de Ejecución

# Uso del programa

Nota: Cada valor que el usuario ingrese debe confirmarse presionando la tecla Enter.

Al iniciar el programa, el usuario debe ingresar su nombre de usuario.

Seleccionar una opción del menú principal escribiendo el número correspondiente:

1  Iniciar juego

2  Ver historial de partidas

3  Salir

Durante el juego, el usuario debe ingresar el valor correspondiente al movimiento del personaje.

# Controles del juego

8 → Arriba

5 → Abajo

4 → Izquierda

6 → Derecha

F → Terminar partida

# Consideraciones

 El jugador inicia con 3 vidas
 
 El objetivo es recolectar todos los premios
 
 El juego registra el historial de partidas
